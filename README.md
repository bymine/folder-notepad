# Folder NotePad - 폴더형 메모장
데스크탑의 폴더와 파일 구조와 유사하게 메모를 생성하고 관리할 수 있는 메모장 앱 입니다.

## About Project
플러터 Getx 상태관리 방식과 SQLite 데이터베이스를 사용하였습니다.


## Installation

```sh
git clone https://github.com/bymine/folder-notepad.git
```
